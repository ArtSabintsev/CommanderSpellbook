Search Tips
  • Type in the name of a single card, a Combo ID, or a Combo Result (see table below) to search the database.
  • Use the Color Filter bar to search for combos within in a specific color combination.
  • Use the Card Filter bar to search for combos with a specific amount of cards.
  • You can view all combos for a specific color by typing in its colloquial color identity name (e.g., Mono-Blue, Dimir, Esper, Sans-Red, WUBRG, etc.)
Special Syntax
What does it do?
Banned Returns combos that contain at least one card that is banned in Commander/EDH.
Spoiled Returns combos that contain at least one card from the an upcoming non-reprint set.

About
The Commander Spellbook project exists to catalog all reasonable Commander/EDH combos and to make them easily available across all modern digital platforms. This community driven project is used to power EDHREC's Combo Feauture.

The database and the source code for the website are completely free and open source under the MIT License. We encourage you to copy this data so that it lives on!
